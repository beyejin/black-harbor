# ⚓ 검은 항구 — 5분 룰북 (쉬운 판)

> 규칙 버전 **2.3-playtest** · 4인 · 8라운드 · 25~35분
> 정식 룰북을 다 읽을 필요 없이, 이것만 읽으면 한 판 할 수 있습니다.

---

## 1. 목표

당신은 항구를 주름잡는 상단의 대표입니다. **8라운드 뒤 점수가 가장 높으면 승리.**

~~~
점수 = 골드 + 남은 상품의 잔존 가치 − 세관 벌점
~~~

골드는 0 미만이 될 수 없지만, 세관 벌점은 최종 점수에서 차감되므로 최종 점수는 음수가 될 수 있다.

돈을 버는 방법은 세 가지입니다.

1. **계약 완료** — 비밀 계약에 적힌 상품을 모아 배송하면 골드를 받는다 (주 수입원)
2. **시세 차익** — 시장에서 싸게 사서 비싸게 판다
3. **밀수** — 계약을 밀수로 배송하면 +4골드 더 받는다. 대신 위험해진다

---

## 2. 내가 가진 것

| 자원 | 시작값 | 설명 |
|---|---:|---|
| 골드 | **24** | 입찰, 구매, 뇌물, 정보상에 사용. 0 밑으로 못 내려감 |
| 비밀 계약 | 3장 | 2장을 골라 **활성**, 1장은 **예비**(첫 계약 완료 시 활성화). 남에게 안 보임 |
| 정보상 사용권 | 1회 | 게임에 딱 한 번, 상대 계약을 엿본다 |
| 의심도 | 0 | 밀수하면 +2. **모두에게 공개** |

중요: **내 골드와 보유 상품은 상대에게 전부 보입니다.** 숨겨지는 건 계약과 아직 공개 안 된 계획뿐입니다. 상대의 골드·상품을 보고 "쟤는 비단 계약이 있겠구나"를 추리하는 게임입니다.

---

## 3. 한 라운드 = 6단계

~~~
① 소식·화물 공개 → ② 정보상 → ③ 경매 → ④ 시장 → ⑤ 계약 배송 → ⑥ 세관
~~~

모든 선택은 **비공개로 작성 → 제출 → 동시 공개**입니다. 제출하면 수정할 수 없습니다. 빨리 클릭할 필요 없습니다.

### ① 항구 소식과 화물 공개

이번 라운드의 이벤트(소식 카드)와 경매에 나올 화물 2개가 공개됩니다.

### ② 정보상 (게임당 1회, 2골드)

상대 한 명 + 상품 하나를 고르면, **그 상대의 활성 계약에 그 상품이 필요한지 YES/NO**로 알려줍니다.
- 정보상 사용 여부는 공개되지만, **조사 대상·질문한 상품·YES/NO 응답은 사용한 사람만 확인합니다.**

### ③ 화물 2개 동시 입찰

화물 2개에 **동시에** 비공개 입찰합니다.

- 두 입찰액의 합 ≤ 내 골드
- 각 화물엔 최소 입찰가가 있음
- **한 사람은 최대 1개만 낙찰.** 둘 다 1등이면 미리 지정한 "선호 화물"을 가져가고, 나머지는 2등에게 넘어감
- 진 사람은 돈을 안 냄. 유찰된 화물의 상품은 다음 라운드 시장에 풀림

> 요령: 상대가 어느 화물을 노리는지 추리해서, 굳이 비싸게 안 사도 되는 쪽에 힘을 빼세요.

### ④ 시장

주문 슬롯 2개. 각 슬롯에 구매 / 판매 / 뇌물 / 패스 중 하나 (수량 1~3).

- **지금 가격에 체결**되고, 시장의 수입/수출 한도를 넘치면 나눠 가짐
- **이번 라운드에 많이 사면 다음 라운드 가격이 오르고, 많이 팔면 내립니다.** 가격은 카드가 아니라 플레이어들이 만듭니다
- **뇌물** (3골드): 내 의심도 −2

### ⑤ 계약 배송 (라운드당 1개)

계약에 적힌 상품을 갖고 있으면 배송할 수 있습니다.

- **합법 배송**: 적힌 골드를 받고 끝. 안전
- **밀수 배송**: +4골드 더 받지만 → 내 **의심도 +2**, 공용 **세관 경계 +1**

### ⑥ 세관

**공용 세관 경계가 4에 도달하면 조사 발생.** 여기가 이 게임의 백미입니다.

- 걸리는 사람은 **마지막에 밀수한 사람이 아니라, 그 순간 의심도가 가장 높은 사람**
- 벌점 = 2 + (의심도 × 2) → 게임 끝에 점수에서 차감. 의심도는 0으로 리셋
- 즉, 내 의심도가 낮으면 **일부러 밀수해서 경계를 4로 채워, 의심도 높은 경쟁자를 조사받게 만들 수 있습니다**

---

## 4. 게임 끝 (8라운드 후)

- 세관 경계가 4 이상이면 일반 조사 먼저, 그 뒤에도 의심도 남은 사람이 있으면 **최종 조사 1회 추가** — "마지막 라운드에 밀수하고 도망"은 안 통합니다
- 남은 상품은 8라운드 시장 결과가 반영된 **최종 시장 가격의 절반(소수점 버림)**으로 계산합니다. 예: 시장 가격이 9골드면 잔존 가치는 4골드
- 완료 못 한 계약 = 0점

---

## 5. 자주 틀리는 규칙 TOP 5

1. **세관은 밀수한 사람을 잡는 게 아니라 의심도 1위를 잡는다** — 남의 밀수에 내가 걸릴 수 있다
2. 경매에서 두 화물 다 1등이어도 **하나만** 가져간다 (선호 화물)
3. 시장 거래는 **이번 라운드 가격**으로 체결된다 — 내가 올린 가격은 다음 라운드에 반영
4. 예비 계약은 **첫 계약을 완료해야** 활성화된다 (정보상 조사에도 안 걸림)
5. 산 상품을 그 라운드에 되팔 수는 없다 (한 상품엔 주문 1개)

| 시작 골드 | 24 |
| 라운드 | 8 |
| 정보상 | 게임당 1회 · 2골드 |
| 경매 화물 | 라운드당 2개 · 1인 최대 1개 낙찰 |
| 시장 주문 | 슬롯 2개 · 수량 1~3 |
| 뇌물 | 3골드 → 의심도 −2 |
| 밀수 | +4골드 · 의심도 +2 · 경계 +1 |
| 세관 경계 한계 | 4 |
| 세관 벌점 | 2 + 의심도×2 |
| 계약 완료 한도 | 라운드 1개 · 게임 3개 |
| 잔존 가치 | 최종 시장 가격 ÷ 2 (버림) |

**라운드 우선순위** (동점·나눠 갖기 처리): 1라운드 시작 좌석은 게임 시드로 무작위 결정되어 공개되고, 매 라운드 한 칸씩 회전합니다.
