# ⚓ 검은 항구 — 플레이어블 데모

이중 봉인 경매 · 플레이어 주도 시장 · 밀수와 세관 조사를 중심으로 한 4인 경제 전략 게임(룰 **2.3-playtest**)의 웹 데모.

- 📖 [5분 쉬운 룰북](docs/rulebook-easy.md)
- 📊 [밸런스 노트 2.2 → 2.3](docs/balance-2.3.md) — 시작 골드 24 상향, 우선순위 시드 오프셋의 근거 데이터

사용자가 좌석 A를 맡고, 상단 B/C/D는 휴리스틱 AI가 플레이한다. 서버 없이 룰북의 판정 로직 전체(경매 승계, 비례 배분, 가격 변동, 세관 조사, 최종 점수)를 클라이언트에서 처리한다.

## 실행

```bash
npm install
npm run dev   # http://localhost:5240
```

## 엔진 검증

```bash
npx tsx scripts/sim.ts   # AI 4인 자동 대전 200시드 × 8라운드 불변식 체크
```

## 구조

- `src/game/engine.ts` — 룰 판정 (경매·시장·계약·세관·점수)
- `src/game/ai.ts` — B/C/D 봇 휴리스틱
- `src/game/data.ts` — 화물 16장, 항구 소식 8장, 계약 12장, 기준 수치
- `src/App.tsx` — 라운드 단계 머신 UI

## 데모에서 생략한 규칙

제한시간 타이머, 잠근 인원수 표시, 재접속 복구 등 실시간 멀티플레이 전용 규칙(§13)은 AI가 즉시 잠그는 구조에서는 의미가 없어 생략했다.
